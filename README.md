# e-sign-verify
Project Title

Secure Digital Organ Donation System using Django & RSA Cryptography

Description

Developed a secure web-based Digital Organ Donation System using Django (Python) to streamline donor registration and organ donation management. The application integrates RSA asymmetric encryption to protect sensitive donor information, ensuring confidentiality and secure data handling. It also implements digital signatures to verify data integrity and authenticity. The system features user authentication, secure data storage, form validation, and an intuitive interface for managing donor records.

Key Features:

Built with Django, Python, HTML, CSS, Bootstrap, and SQLite
Implemented RSA encryption to secure sensitive donor information
Used digital signatures to verify data integrity
User registration, authentication, and secure donor management
Responsive web interface with organized backend architecture
Focused on cybersecurity best practices for healthcare data protection

Skills: Python · Django · Cryptography · RSA Encryption · HTML · CSS · Bootstrap · SQLite · Cybersecurity · Web Development
